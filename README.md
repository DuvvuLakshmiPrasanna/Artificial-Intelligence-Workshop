## 📖 About

This repository brings together every lab and project completed throughout the workshop into a single, organized monorepo. Each folder is a self-contained mini-project exploring a different facet of applied AI — from environment setup and prompt-pattern drills to resume scoring, research automation, and workflow orchestration with n8n.

## 🗂️ Projects

| # | Project | Description |
|---|---------|-------------|
| 1A | [AI Playground Lab](./AI-Playground-Lab-1A) | An introductory sandbox for experimenting with core AI/LLM concepts and first API calls. |
| 1B | [Toolkit Setup Lab](./Toolkit-Setup-Lab-1B) | Environment and toolchain setup — installing and configuring the libraries and tools used across the workshop. |
| 2A | [Six Pattern Drills Lab](./Six-Pattern-Drills-Lab-2A) | Practice drills covering six core prompt-engineering patterns for reliable LLM outputs. |
| 2B | [JSON Résumé Extractor](./JSON-R-sum-Extractor-2B) | Parses unstructured résumé text and extracts structured data into clean JSON. |
| 3A | [3-Step Verification Chain](./3-step-Verification-Chain-3A) | A chained-prompting pipeline that verifies and refines AI-generated answers across three stages. |
| 3B | [Placement Cell AI Policy](./Placement-Cell-AI-Policy-3B) | An AI usage policy/guideline document designed for a college placement cell. |
| 4A | [Research Deck Sprint](./Research-Deck-Sprint--4A) | Rapid AI-assisted research summarized into a presentation-ready deck. |
| 4B | [n8n Daily News Digest & Job Notifications](./n8n-Daily-News-Digest--JobNotifications--4B) | An n8n automation workflow that delivers daily news digests and job alert notifications. |
| 5A | [Resume Scorer](./Resume-Scorer-5A) | Scores and evaluates résumés against job criteria using AI-driven analysis. |
| 5B | [Hugging Face Pulls](./Hugging-Face-Pulls-5B) | Explores pulling and using pretrained models/datasets from the Hugging Face Hub. |

## 🚀 Getting Started

Clone the full repository:

```bash
git clone https://github.com/DuvvuLakshmiPrasanna/Artificial-Intelligence-Workshop.git
cd Artificial-Intelligence-Workshop
```

Each sub-folder is independent — `cd` into the one you're interested in and follow any notebook or script inside it. Most projects are Jupyter notebooks; open them with:

```bash
jupyter notebook
```

### Prerequisites

- Python 3.9+
- Jupyter Notebook / JupyterLab
- `pip install -r requirements.txt` (if a project includes one)
- An API key for your LLM provider of choice (where applicable)
- [n8n](https://n8n.io/) (only required for the workflow-automation project, 4B)

## 🧰 Tech Stack

- **Jupyter Notebook** — primary format for labs and experiments
- **Python** — supporting scripts and utilities
- **n8n** — no-code workflow automation
- **Hugging Face** — pretrained models and datasets
