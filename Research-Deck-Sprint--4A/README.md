# Day 4 — Research + Deck Sprint

## Company Selected

**Infosys**

## Objective

Create a placement-preparation brief and presentation deck for B.Tech students within a strict 45-minute timebox using free AI-powered research and content-generation tools.

## Tools Used

* Perplexity AI (Research)
* Google AI Studio with Grounding (Fact Verification)
* Gamma (Presentation Generation)
* Google Docs (Brief Formatting)
* GitHub (Version Control & Submission)

## Deliverables

### 1. Placement Preparation Brief

**File:** `Day4_Infosys_brief.pdf`

Contents:

* Company Overview
* Hiring Process
* Technical Stack
* Recent News
* Eligibility Criteria
* Preparation Tips
* Sources & References

### 2. Placement Preparation Deck

**File:** `Day4_Infosys_deck.pdf`

Contents:

* Company Overview
* Why Join Infosys
* Hiring Process
* Technical Skills Required
* Eligibility Criteria
* Recent Industry Updates
* Preparation Strategy
* Action Plan

## Workflow Followed

### Research Phase

1. Planned 5 focused research questions before using Perplexity.
2. Collected answers and source URLs.
3. Cross-checked important information using grounded search.

### Brief Creation

1. Consolidated research findings.
2. Generated a structured placement-preparation brief.
3. Added citations and references.
4. Exported as PDF.

### Deck Creation

1. Generated an 8-slide presentation using Gamma.
2. Reviewed every slide manually.
3. Verified statistics and eligibility information.
4. Edited the cover and CTA slides.
5. Exported the final deck as PDF.

## Key Learnings

* AI tools significantly reduce research and content creation time.
* Generated content must always be fact-checked before publication.
* Timeboxing improves productivity and decision-making.
* Human review remains essential for accuracy and quality.
